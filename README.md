# 🧠 Alzheimer's Disease Detection Using Deep Learning

An AI-powered web application that predicts the stage of Alzheimer's disease from brain MRI images using a Deep Learning model integrated with the Django web framework.

## 📌 Project Overview

This project is designed to assist in the early detection of Alzheimer's disease by analyzing MRI brain scan images. Users can securely log in, upload an MRI image, and receive the predicted Alzheimer's stage along with the model's confidence score.

The application combines Deep Learning with a user-friendly web interface to demonstrate the practical use of Artificial Intelligence in healthcare.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/alzheimers-disease-detection.git
cd alzheimers-disease-detection
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### macOS/Linux

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Django Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
